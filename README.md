## Install ⬇️
Tuist 설치
```shell
curl -Ls https://install.tuist.io | bash
```

프로젝트 오픈
```shell
git clone https://github.com/Newon-universe/AppStore-mock-coding.git
cd 클론한 폴더
tuist fetch
tuist generate
```
